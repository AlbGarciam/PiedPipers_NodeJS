Route serving local searching process.
Authentication

This route can use JWT verification. If you don't have the JWT you need to sign in with a valid user

Query Parameters:
Name Type Description
name string Local's name
price number Local's max price
lat number Profile's latitude. Must be present if long or maxDistance exists
long number Profile's longitude. Must be present if lat or maxDistance exists
maxDistance number Profile's search radius (in km). Must be present if long or lat exists
limit number Maximun number of profiles. By default it takes 10
offset number Skips profiles

Route:
MethodPath
GET serach/profile
Route serving profile searching process. This this API is authenticated, it won't response the current profile. If not, current profile can be added to the response
Authentication

This route can use JWT verification. If you don't have the JWT you need to sign in with a valid user

Query Parameters:
Name Type Description
name string Profile's name
instruments string Profile's skills separated by commas
lat number Profile's latitude. Must be present if long or maxDistance exists
long number Profile's longitude. Must be present if lat or maxDistance exists
maxDistance number Profile's search radius (in km). Must be present if long or lat exists
friendlyLocation string Profile's friendly location.
limit number Maximun number of profiles. By default it takes 10
offset number Skips profiles
